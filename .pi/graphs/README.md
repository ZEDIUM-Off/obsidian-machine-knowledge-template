# Workspace graphs

Les graphes de ce dossier orchestrent les workflows transverses : inventaire, focus, migrations, dépendances et handoffs.

Ils peuvent maintenir une vue dans `.agents/state/`, mais toute connaissance durable est proposée dans le bundle canonique propriétaire. Les artefacts d'exécution restent sous `.pi-subagents/` et ne sont jamais indexés par Obsidian.
